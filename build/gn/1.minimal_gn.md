# Gn 最小配置

一个 GN 构建项目至少包括

```
.
├── BUILD.gn
├── build
│   ├── BUILD.gn
│   ├── BUILDCONFIG.gn
│   └── toolchain
│       └── BUILD.gn
├── 代码
|  ...
```

`BUILD.gn` 是构建的入口点。

## 依赖

### group 

在GN中，组只是没有编译或链接的依赖项的集合

```
group("tools") {
  deps = [
    # This will expand to the name "//tutorial:tutorial" which is the full name
    # of our new target. Run "gn help labels" for more.
    "//tutorial",
  ]
}
```
